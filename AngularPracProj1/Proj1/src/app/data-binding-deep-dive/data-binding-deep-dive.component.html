View Encapsulation :
Angular has View Encapsulation which applys css to only that component in which it is written. Angular gives random attribute names to each component and its elements to replicate shadow dom technology. But we can override this by setting the Encapsulation propeerty in metadata of the component to encapsulation: ViewEncapsulation.None. So now the css will get applied globally. By default it is as encapsulation: ViewEncapsulation.Emulated. We can also set the encapsulation property to encapsulation: ViewEncapsulation.Native which behaves like Emulated but only on those browsers which support shadow dom technology.
